Medical certificate of cause of death service

You can use this service if you are an attending practitioner or a medical examiner.

If you are an attending practitioner you can use this service to:

  • create an MCCD
  • medical information relating to the cause of death
  • send the MCCD to the registrar

If you are a medical examiner you can use this service to:

  • review an MCCD
  • accept or reject an MCCD
  • return the MCCD back to the attending practitioner to amend

Before you register an account

You'll need your work address, GMC number and qualifications.

It takes about 10 minutes to register an account with the MCCD service.

Before you create an MCCD

You’ll need to register an account with the MCCD service.

You’ll need personal details of the deceased person, medical information relating to the cause of death.
